Mandelbrot

El conjunto de Mandelbrot es el conjunto fractal más estudiado. Se define en el plano complejo fijando un número complejo c cualquiera. A partir de c, se construye una sucesión por recursión:

Z0 = 0 ∈ ℂ Zn+1 = Zn2 + c

El conjunto de Mandelbrot es el conjunto de números complejos c para los que la sucesión no diverge.

En este proyecto, se muestra una representación interactiva del conjunto de Mandelbrot, para explorar el conjunto simplemente haz "click" y se ampliará la zona marcada.
